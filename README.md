# Bloom
A simple and user-friendly to-do list application that helps you keep track of tasks. Users can add, edit, mark as complete, and delete tasks. 

## Features
- Add new tasks
- check in-progress tasks
- Mark tasks as complete
- Delete tasks you no longer need
- Real-time progress wheel

## Technologies Used
- HTML, CSS, JavaScript

## How to Run
1. Clone the repository.
2. Open `index.html` in your browser
